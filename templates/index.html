Instagram Bot

Instructions

  • Compile your list of Instagram accounts you want to work on
  • Save it as a txt file
  • Click on the upload button to upload the File
  • Click on upload to start
  • Wait for the application to finish running
  • A path to the finished file will be shown on screen
  • Go to the path and view the operation that were completed successfully

Note:

Only txt files are accepted

Any account that was unsuccessfull should be complied into another text file

The file should be uploaded again
